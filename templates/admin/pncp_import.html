Importar atas de registro de preço (PNCP)

O Comprasnet em si não oferece uma API pública única para “todas as atas”. O cadastro nacional aberto é o PNCP (API de consulta): endpoint GET /v1/atas por período de vigência (datas no formato yyyymmdd).

Aqui buscamos atas vigent no intervalo que você informar e, se marcar o filtro, mantemos só as com início de vigência no ano escolhido (ex.: 2026). A API não traz preço de itens — só objeto, órgão e vigência; cada ata vira um item no catálogo com valor R$ 0,00 para você revisar no painel.

Atenção: intervalos amplos geram muitas páginas (até 500 registros por página). Limite “máx. páginas” para testar antes. Use com responsabilidade para não sobrecarregar o servidor público.

A importação pode levar vários minutos. Não feche o navegador até concluir.
