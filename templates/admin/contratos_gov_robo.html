3 · Busca híbrida (PNCP + Contratos)

Opcional

Usa o período e filtros PNCP da seção 1 + filtros Contratos abaixo.

  • PNCP → possibilidadeAdesao + filtros do portal
  • Contratos → itens com “Aceita adesão = Sim” + CNPJ fornecedor
  • Se houver CNPJ de fornecedor, pula o PNCP (API não filtra fornecedor)
Importar PNCP (catálogo)

Busca principal (PNCP)

  • Filtros iguais ao portal /app/atas
  • Permite Adesão = Sim → possibilidadeAdesao na API
  • Esfera / UF → cache local + cadastro do órgão

API: GET /consulta/v1/atas · Esfera/UF usam cache local.

Outras buscas

  • Contratos — confirma adesão por item
  • Híbrido — PNCP + Contratos em sequência

Consulta incremental

Use Atualizações no período para monitorar só atas novas/alteradas — recomendado pelo PNCP para robôs.
